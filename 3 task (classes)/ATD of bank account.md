# АТД «Клиент банка» (BankUser)

## • Атрибуты

- **`full_name`** — полное имя клиента (строка)  
- **`bank_code`** — код банковского филиала (строка)  
- **`current_balance`** — текущая сумма на счете (число с плавающей точкой)

---

## • Операции

### – Конструктор

- **Входные параметры**: `client_name` (строка), `` (строка), `opening_deposit` (число с плавающей точкой)  
- **Предусловия**: `opening_deposit >= 0`  
- **Действие**: инициализация атрибутов объекта заданными значениями  
- **Возвращаемое значение**: —  
- **Постусловия**: создан новый клиент с указанными именем, кодом филиала и начальным балансом  
- **Исключения**: `std::invalid_argument`, если `opening_deposit < 0`

---

### – Получение текущего баланса

- **Входные параметры**: `` (строка)  
- **Предусловия**: отсутствуют  
- **Действие**: возвращает значение `current_balance`  
- **Возвращаемое значение**: текущий баланс (число с плавающей точкой)  
- **Постусловия**: состояние объекта остаётся неизменным

---

### – Получение имени клиента

- **Входные параметры**: `` (строка)  
- **Предусловия**: отсутствуют  
- **Действие**: возвращает значение `full_name`  
- **Возвращаемое значение**: имя клиента (строка)  
- **Постусловия**: состояние объекта остаётся неизменным

---

### – Внесение средств на счет

- **Входные параметры**: `sum` (число с плавающей точкой), `` (строка)  
- **Предусловия**: `sum > 0`  
- **Действие**: увеличивает `current_balance` на величину `sum`  
- **Возвращаемое значение**: —  
- **Постусловия**: баланс увеличен на `sum`  
- **Исключения**: `std::invalid_argument`, если `sum <= 0`

---

### – Снятие средств со счета

- **Входные параметры**: `sum` (число с плавающей точкой), `` (строка)  
- **Предусловия**:  
  - `sum > 0`  
  - `current_balance >= sum`  
- **Действие**: уменьшает `current_balance` на величину `sum`  
- **Возвращаемое значение**: —  
- **Постусловия**: баланс уменьшен на `sum`  
- **Исключения**:  
  - `std::invalid_argument`, если `sum <= 0`  
  - `std::runtime_error`, если на счете недостаточно средств

---

### – Перевод средств на другой счет

- **Входные параметры**: `sum` (число с плавающей точкой), `` (строка), `recipient_account` (ссылка на другой объект BankUser)  
- **Предусловия**:  
  - `sum > 0`  
  - `current_balance >= sum`  
  - `recipient_account` не равен текущему объекту  
- **Действие**:  
  1. Уменьшает `current_balance` текущего объекта на `sum`  
  2. Увеличивает `current_balance` объекта `recipient_account` на `sum`  
- **Возвращаемое значение**: —  
- **Постусловия**:  
  - Баланс текущего клиента уменьшен на `sum`  
  - Баланс получателя увеличен на `sum`  
- **Исключения**:  
  - `std::invalid_argument`, если `sum <= 0`  
  - `std::runtime_error`, если на счете недостаточно средств  
  - `std::invalid_argument`, если `recipient_account` равен текущему объекту

### – Экспорт данных в файл (режим добавления)

- **Входные параметры**: `path` (строка) — путь к целевому файлу  
- **Предусловия**: файл должен быть доступен для записи  
- **Действие**: открывает файл для дозаписи и записывает в него строку с данными клиента (`full_name`, `bank_code`, `current_balance`)  
- **Возвращаемое значение**: —

- **Исключения**: `std::ios_base::failure`, если операция открытия/записи в файл не удалась

---

- **Действие**: считывает данные из потока в порядке: `full_name`, `bank_code`, `current_balance`  
- **Возвращаемое значение**: —  
- **Постусловия**: атрибуты объекта заменены считанными значениями  
- **Исключения**: `std::ios_base::failure`, если операция чтения из потока невозможна
